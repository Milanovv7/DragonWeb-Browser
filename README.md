# DragonWeb-Browser
To get DragonWeb you need Windows and you must have Chrome installed with Ignore X-Frame headers. Then, right click on your desktop>New>Shortcut and put this prompt in that window: "C:\Program Files\Google\Chrome\Application\chrome.exe" --app="file:///C:/Users/matej/OneDrive/Desktop/App%20Development/DragonWeb/Emergent%20_%20Fullstack%20App.html"
