# DragonWeb-Browser


DragonWeb v1.0.1 is here.

Here's what's new:

• Errors when switching between tabs bug fixed.

• Errors when opening the app bug fixed.

• Now even smoother performance.




Make sure that your computer meets its OS requirement:

• macOS 10.12 Sierra or later

• Windows 7 or later

• Linux (depends on the distro)

• ChromeOS version 72 or later






Here's how to get it!





Open Google Chrome and install the Ignore X-Frame headers extension from the Chrome Web Store, which you can do by searching up Chrome Web Store in your Chrome search bar. In Chrome Web Store search for Ignore X-Frame headers. Click on the first extension that pops up and click on Add to Chrome. Then, click on this URL: https://milanovv7.github.io/DragonWeb-Online-Demo/


You'll be taken to the web version of the browser in a new Chrome tab. From here you can test it before actually installing it. If you wanna install it, while you're here click on the 3 dots menu in the top right corner, hover over Cast, save, and share. Click on Download page as app.

If you want you can even download the code for DragonWeb by dowloading the index.html & Emergent_Fullstack App_Files. You can view the code for yourself in Visual Studio Code and edit the code if you want.

Enjoy your browsing.
