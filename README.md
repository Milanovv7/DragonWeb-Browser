Here's how to get it!




New: You can now download DragonWeb with Microsoft Edge. Just Click on the "..." in the top right corner of your screen, click on "Extensions", click on "Manage extensions" and in the new tab that opened enable Developer mode and enable Allow extensions from other stores. You can now follow the steps:





Open Google Chrome (or Microsoft Edge) and install the Ignore X-Frame headers extension from the Chrome Web Store, which you can do by searching up Chrome Web Store in your Chrome (or clicking on https://chromewebstore.google.com/ and then installing the extension if you're on Edge) search bar. In Chrome Web Store search for Ignore X-Frame headers. Click on the first extension that pops up and click on Add to Chrome. Then, click on this URL: https://milanovv7.github.io/DragonWeb-Online-Demo/.

You'll be taken to the web version of the browser in a new Chrome/Edge tab. From here you can test it before actually installing it. If you wanna install it, while you're here click on the 3 dots menu in the top right corner, hover over Cast, save, and share. Click on Download page as app. If you're on Microsoft Edge you're again gonna click on the 3 dots but click on "Apps" and then on "Install this Site as App".

If you want you can even download the code for DragonWeb by dowloading the index.html & index_files. You can view the code for yourself in Visual Studio Code and edit the code if you want.

Enjoy your browsing.
